# Netflix Data Analysis (Interactive Dashboard using Power BI)
## Project Objective

This project analyzes Netflix movies and TV shows data to understand content trends, ratings, genres, countries, and release year growth using an interactive Power BI dashboard.
## Dataset used
- <a  href="https://github.com/Umar2410/Data-Analysis_dashboard/blob/main/netflix_titles.csv"><dataset</a>
 
## Key Questions (KPIs)

Total number of shows on Netflix

Distribution of Movies vs TV Shows

Most common content ratings

Top genres by total shows

Content distribution by country

Growth of shows over release years

Start year and end year of content availability
